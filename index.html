HTML

  • HTML stands for Hypertext markup language.
  • HTML is the primary language used to create web pages.
  • HTML also tells a web page how to display text and images.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS defines appearance and style of a website.

GIT

  • Git is a open-source version control system.
  • Git can track changes in a source code and enable people to work together.
  • Git can handle from small to large projects efficiently.

JavaScript

  • JavaScript is used for 99% of websites for web page behavior.
  • JavaScript allows us to store useful values inside variables.
  • Functions are blocks of code that contain instructions for a certain task.
